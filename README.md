## 🚀 快速开始

### 安装依赖

\`\`\`bash
npm install
\`\`\`

### 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

### 构建生产版本

\`\`\`bash
npm run build

\`\`\`

## 🎯 自定义配置

### 修改个人信息

在相应组件中更新以下内容：

- `Hero.tsx` - 姓名、职业、介绍
- `About.tsx` - 个人故事、经历
- `Skills.tsx` - 技能和工具
- `Projects.tsx` - 项目展示
- `Contact.tsx` - 联系方式

### 修改主题色彩

在 `globals.css` 的 `:root` 中修改 CSS 变量：
\`\`\`css
:root {
  --neon-purple: #a855f7;
  --neon-pink: #f472b6;
  /* 其他颜色变量 */
}
\`\`\`
